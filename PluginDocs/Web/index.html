# 前端

对于每个插件，在其文件夹下都有src子文件夹，此文件夹内部有两个文件：index.html与config.html。

  • index.html 为插件页面的入口点
  • config.html 为插件的配置页面

编写插件页面需要你有基本的HTML与JavaScript知识，HiperLink内置了Vuetifyopen in new window 框架，因此您可以直接开始！
